# UrlOpener

UrlOpener — Android-приложение для быстрого открытия и хранения ссылок.

## Возможности

- открытие введённой ссылки во внешнем приложении;
- сохранение ссылок с именем;
- группировка сохранённых ссылок;
- добавление, редактирование и удаление групп;
- добавление, редактирование и удаление ссылок внутри групп;
- подтверждение удаления ссылок и групп;
- подтверждение открытия сохранённой ссылки;
- светлая и тёмная темы;
- работа от края до края экрана.

## Стек

- Kotlin
- Jetpack Compose
- Coroutines
- Room
- DataStore
- Material 3

## Сборка

Для сборки отладочной версии:

```bash
./gradlew :app:assembleDebug
```

Для запуска модульных тестов:

```bash
./gradlew :app:testDebugUnitTest
```

Для сборки инструментальных тестов:

```bash
./gradlew :app:assembleDebugAndroidTest
```
